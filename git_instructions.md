# Git-инструкции
*Создаем изменения в файле и добавляем его в отслеживаемые:* 
**git add <название файла>**

*Добавляем в отслеживаемые все файлы:*
**git add .**

*Сделать коммит:*
**git commit -m "Комментарий к изменению""**

*Просмотр истории коммитов:*
**git log**

*Вывод последнего коммита:*
**Git show HEAD**
